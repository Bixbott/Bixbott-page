Bixbott Ecosystem

Một tổ chức AI agent được chia module rõ ràng.

Mỗi phần trong Bixbott có trách nhiệm riêng, giúp dự án dễ phát triển, dễ thay thế và dễ mở rộng khi lớn hơn.

01

Management

Quản trị cấu hình, người dùng, quyền truy cập, trạng thái hệ thống và vận hành tổng thể.

02

Agent

Thực thi tác vụ tự động, kết nối công cụ, chạy workflow và phản hồi theo ngữ cảnh.

03

Council

Điều phối nhiều agent, hỗ trợ lập kế hoạch, đánh giá phương án và ra quyết định.

04

Core

Chứa logic nền tảng, kiểu dữ liệu, API nội bộ và quy tắc nghiệp vụ quan trọng.

05

CLI

Công cụ dòng lệnh cho developer/operator để chạy lệnh, deploy, debug và tự động hóa.

06

Dashboard

Giao diện web hiện đại để theo dõi agent, xem logs, tạo nhiệm vụ và quan sát tiến độ.

07

Docs

Tài liệu hướng dẫn cài đặt, kiến trúc, API, ví dụ sử dụng và quy chuẩn đóng góp cho cộng đồng.

System Design

Thiết kế theo lớp, tập trung vào khả năng mở rộng.

Dashboard và CLI là lớp tương tác. Management điều hành trạng thái. Council lập kế hoạch và phân phối nhiệm vụ. Agent thực thi, còn Core giữ toàn bộ logic quan trọng để các phần còn lại dùng lại nhất quán.
