# Contributing to Bixbott Organization Hub

Thank you for helping improve Bixbott.

## Principles

- Keep the website aligned with the canonical [`Bixbott/.github`](https://github.com/Bixbott/.github) profile repository.
- Prefer clear Vietnamese copy for visitor-facing landing page content.
- Keep static assets lightweight and dependency-free unless the project adopts a build system.
- Document new architecture decisions in `docs/`.

## Local checks

```bash
python3 -m http.server 4173
```

Then open `http://127.0.0.1:4173/` and verify the landing page, links, and responsive layout.
